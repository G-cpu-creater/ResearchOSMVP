## Step 1: Go to Vercel Dashboard

1. Open: https://vercel.com/dashboard
2. Click on your **ElctrDc** project
3. Click **Settings** (top menu)
4. Click **Environment Variables** (left sidebar)

---

## Step 2: Add Environment Variables

Add these 3 environment variables:

### 1. GROQ_API_KEY (Required for AI features)
```
Key: GROQ_API_KEY
Value: your_groq_api_key_here
Environment: Production, Preview, Development (check all 3)
```

### 2. BLOB_READ_WRITE_TOKEN (Required for file uploads)
```
Key: BLOB_READ_WRITE_TOKEN
Value: (This will be auto-generated by Vercel when you first use Blob storage)
Environment: Production, Preview, Development (check all 3)

NOTE: Leave this empty for now. Vercel will create it automatically.
```

### 3. DATABASE_URL (Required for data storage)
```
Key: DATABASE_URL
Value: your_database_connection_string
Environment: Production, Preview, Development (check all 3)

NOTE: Use your existing database URL or create a new one.
```

---

## Step 3: Save and Redeploy

After adding the environment variables:

1. Click **Save** on each variable
2. Go back to **Deployments** tab
3. Click **Redeploy** on the latest deployment

OR just wait for the automatic deployment from GitHub!
